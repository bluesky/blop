Installation

Running Tutorials

You can run the tutorials in Jupyter Lab locally or in a browser using Binder:

  • Binder Blop Tutorials

If you are using Pixi, you can start a local Jupyter Lab instance:

$ pixi run start-jupyter

Or convert the tutorials to ipynb format and use your preferred tool:

$ jupytext --to ipynb docs/source/tutorials/*.md

Developer Installation

We recommend using Pixi to manage your development environments. If you prefer not to use Pixi, you can view the configuration in pixi.toml and create your own environment.

Static Checks

For linting, formatting, and static code analysis:

$ pixi run check

Run Tests

For running the tests:

$ pixi run unit-tests
$ pixi run test-docs

Build Documentation

For building this documentation:

$ pixi run build-docs
